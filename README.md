# Taoist CIO Market Intelligence Dashboard

A five-layer financial intelligence dashboard for a Whole Wealth Advisor practice.

## Structure

- `src/` — Python modules
- `data/` — Cached data and last_readings.json
- `config/` — Thresholds and ticker configs
- `logs/` — Error logs

## Setup

```bash
source venv/bin/activate
pip install -r requirements.txt
```
